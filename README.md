## 如何选择合适的线路

很多新手最纠结的就是选哪种带宽。简单说一下三种线路的区别。

**CN2优化带宽** 是电信的精品网络，也叫中国电信下一代承载网。特点是延迟低、速度快、稳定性好，特别适合国内用户访问。缺点是带宽相对贵一点，而且带宽量一般不会给特别大。如果你的用户主要在国内，首选CN2优化。

**全向带宽** 是混合线路，国内外都能用，但不走CN2专线。延迟会比优化带宽稍高，速度也会慢一些，但带宽量大，价格便宜。适合国内外混合业务，或者对延迟要求不那么苛刻的场景。

**国际带宽** 走的是纯国际BGP线路，不经过国内骨干网。如果你的业务主要面向海外用户，或者需要大带宽跑流量，国际带宽是最好的选择。国内访问的话延迟会比较高，不太适合面向大陆用户的业务。

具体选哪个，还是得看你的业务需求。如果实在拿不准，可以先买个便宜的VPS测试一下，看看哪条线路效果最好，再决定租独立服务器。
